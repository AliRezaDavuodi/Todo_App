پروژه ۱: ساخت Todo List API ساده با Node.js و Express (مقدماتی)
هدف کلی: آشنایی با مفاهیم پایه بکند (راه‌اندازی سرور، REST API، کار با دیتابیس، Auth ساده).

زمان‌بندی: ۲ هفته (روزانه ۲ ساعت)
روز ۱-۲: راه‌اندازی پروژه با Express.js

ایجاد سرور ساده و آشنایی با روتینگ (GET /, POST /todos).

مهارت: کار با Express و Middlewareها.

روز ۳-۴: اتصال به دیتابیس (MongoDB با Mongoose)

نصب MongoDB و ایجاد مدل Todo با فیلدهای title, completed, createdAt.

مهارت: کار با NoSQL و طراحی Schema.

روز ۵-۷: ساخت CRUD کامل برای Todoها

اضافه کردن endpoints برای GET /todos, GET /todos/:id, PUT /todos/:id, DELETE /todos/:id.

مهارت: پیاده‌سازی REST API استاندارد.

روز ۸-۱۰: افزودن احراز هویت ساده با JWT

ساخت روت /auth/signup و /auth/login برای تولید توکن.

اضافه کردن Middleware برای بررسی توکن در روت‌های Todo.

مهارت: مدیریت Auth و Security پایه.

روز ۱۱-۱۴: تست API با Postman و رفع باگ

تست تمام endpointها و مستندسازی API با Swagger.

مهارت: تست دستی API و مستندسازی.

دیتابیس: MongoDB (رایگان روی Atlas).

کتابخانه‌ها: Express, Mongoose, JWT, Swagger.
